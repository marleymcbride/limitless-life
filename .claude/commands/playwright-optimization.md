# Playwright Sales Page Optimization

Quickly test and optimize the Limitless Life sales page using Playwright automation.

## Usage

```bash
/playwright-optimization
```

## What This Command Does

This command automatically executes a comprehensive Playwright testing workflow:

1. **Performance Analysis**
   - Navigate to the sales page
   - Check console errors and warnings
   - Analyze network requests for performance bottlenecks
   - Measure Core Web Vitals impact

2. **Responsive Testing**
   - Desktop (1440x900) - Full page screenshot
   - Tablet (768x1024) - Layout verification
   - Mobile (375x812) - Touch optimization check

3. **Conversion Testing**
   - Test all CTA buttons and links
   - Verify application form flow
   - Check button responsiveness
   - Validate mobile touch targets

4. **Video Player Testing**
   - VSL loading performance
   - Player state verification
   - Error fallback testing
   - Mobile video optimization

5. **Error Monitoring**
   - React hydration mismatch detection
   - JavaScript error analysis
   - Network failure handling
   - Edge case testing

## Automated Output

The command provides:

- **Performance metrics** from browser automation
- **Screenshot evidence** for visual testing
- **Console error analysis** with specific fixes
- **Responsive testing results** across devices
- **Conversion funnel verification**
- **Actionable optimization recommendations**

## When to Use

- **After any UI changes** to ensure no regressions
- **Before deploying** to production
- **When troubleshooting** performance issues
- **For regular optimization** checks
- **When testing new features** or components

## Integration

This command integrates with the sales-page optimization agent for deeper analysis when critical issues are detected.
